# tasks-board-angular

Приложение для управления списком задач

Реализовано:
1. Простые формы входа и регистрации. Сущность пользователя с полями: id и name, password с использованием bscrypt. Для пользователей Ivan и Maria пароль 12345.
2. Таски содержат следующие поля: id, name, description, status, estimate, assignee, reporter, labels, sprint, priority. Есть экран списка задач и доска задач. По клику на задачу на обоих экранах показываются детали с помощью анимации. Формы создания и редактирования задач в модальном окне.
3. drag and drop функциональность для перетаскивания задач между колонками статусов на доске.
4. На обоих экранах есть фильтрация по различным полям, на странице списка дополнительно есть поиск по названию. При смене страницы поля очищаются.
5. Страница статистики текущего пользователя с таблицей по спринтам, затраченного времени и графиками трех наиболее результативных спринтов
6. Поддержана локализация полей
